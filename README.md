# CapstoneProject
Capstone project exploring group cohesion prediction using multimodal deep learning and large language models (LLMs) with structured feature representations.

This repository contains the implementation code for my capstone project in python.

## Structure
- Step1: dataset index built
- Step2: Audio-based ML models
- Step3: Audio-visual ML models
- Step4: Multimodal ML models
- Step5: JSON construction for LLM features
- Step6: LLM Experiments
- Step7: Additional ML experiments to compare with LLM

## Notes

- Scripts correspond to experiments discussed in the report
- Each file contains a header explaining its purpose
